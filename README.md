# Stock Price Tracker

Stock Price Tracker is backend system that integrates with the Alpha Vantage API, collects
and aggregates real time stock price data.

## Project setup

In order to install all neccesary dependencies run 

php artisan serve

To run migrations and seed initial data run

php artisan migrate:fresh --seed
